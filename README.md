# Cross-Asset Market Analytics Dashboard

## Project Overview

This project is an end-to-end financial analytics dashboard built using Python, SQL Server, and Power BI to analyze the behavior of multiple asset classes across changing market conditions.

The dashboard explores:

* Asset price performance
* Risk vs return relationships
* Historical drawdown comparisons
* Rolling market volatility
* Major market and policy events

## Asset Classes Included

* SPY — U.S. Equities
* QQQ — Growth Equities
* GLD — Gold
* DBC — Broad Commodities
* USO — Crude Oil
* TLT — Long-Term U.S. Treasuries

## Tech Stack

* Python (Pandas, NumPy, yFinance)
* SQL Server
* Power BI

## Dashboard Features

* Interactive slicers and filtering
* Cross-asset performance tracking
* Risk vs return analysis
* Historical drawdown comparison
* Rolling market volatility analysis
* Macro and policy event tracking

## Project Workflow

Python → SQL Server → Power BI
